📊 專題名稱：我的作品 (My Portfolio Project)
這是一個結合了 資料庫管理 與 數據視覺化 的網頁專題。本專案旨在透過 SQL Server 進行後端數據儲存與清洗，並利用 Power BI 產出具備洞察力的互動式報表，最後整合於網頁平台展示。

🛠 使用技術
本專案整合了以下開發工具與技術：

版本控制與託管: GitHub (用於代碼管理與專案維護)

數據可視化: Power BI (開發互動式儀表板)

開發環境: Visual Studio Code (VS Code)

資料庫系統: SQL Server (負責數據儲存、T-SQL 查詢優化)

前端展示: HTML5 / CSS3 (網頁介面佈局)

✨ 核心功能
數據整合: 從 SQL Server 提取原始數據，進行 ETL (提取、轉換、載入) 處理。

互動式報表: 使用 Power BI 製作動態圖表，支援篩選器、鑽取 (Drill-down) 功能。

網頁整合: 將數據分析結果嵌入至網頁，方便使用者隨時瀏覽。

響應式設計: 確保作品在不同裝置上都能正常顯示。

🚀 快速上手
1. 資料庫建置
在 SQL Server 中執行 sql/ 資料夾下的腳本以建立資料庫架構與導入初始數據。

2. 查看數據報表
使用 Power BI Desktop 打開 reports/ 下的檔案，並更新資料來源連接至您的 SQL 實例。

3. 運行網頁
透過 VS Code 開啟 web/ 資料夾，並使用 Live Server 擴充功能啟動網頁預覽。
